# Hybrid\\Theme

Hybrid Theme is a collection of helper functions and filters to aid in theme development.

## Requirements

* WordPress 4.9+.
* PHP 5.6+ (preferably 7+).
* [Composer](https://getcomposer.org/) for managing PHP dependencies.
